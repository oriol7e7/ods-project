Empresa sostenible: GitHub

Imatge oficial de GitHub amb Sustainable Development Goals
Accions pel Medi Ambient (Què fan?)

GitHub no només es centra en reduir el consum dels seus propis servidors, sinó que empodera la seva comunitat d'usuaris per crear Green Software (Programari Verd).

Han tingut accions clau com:
Green Software Foundation Amb l'objectiu de reduir les emissions del sector IT.
Green Software Directory Creant un directori obert amb eines i projectes de codi lliure.
IA per a bens socials Proporcionant accés a organitzacions a la seva IA, GitHub Copilot permetent crear projectes mes eficientment.

Grups d'interès

Les iniciatives de GitHub connecten amb diferents col·lectius essencials:

Programadors Als quals se'ls proporciona el "Climate Action Plan" i eines per programar de manera eficient.
ONGs Entitats com The Ocean Cleanup utilitzen la infraestructura de GitHub i ofereixen plans gratuits per a aquestes entitats.
Socis corporatius Empreses com Microsoft que col·laboren a la Green Software Foundation.
